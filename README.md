
MERN Finance Dashboard with Machine Learning Predictions

This project is a full-stack MERN finance dashboard application that integrates machine learning–based predictions to provide data-driven financial insights.
The frontend is built using Vite as the development environment for fast builds and hot module replacement. Redux Toolkit is used for global state management, Material UI (MUI) provides a consistent and responsive component library, and Recharts is utilized to visualize financial data through interactive charts.
The backend is powered by Node.js and Express.js, exposing RESTful APIs to manage data and business logic. MongoDB is used as the database for persistent storage of financial and predictive data. The application follows a modular and scalable architecture suitable for real-world production use.

Tech Stack

Frontend: Vite, React, Redux Toolkit, Material UI, Recharts
Backend: Node.js, Express.js
Database: MongoDB
Machine Learning: Predictive models for financial insights


Dashboard Financiero MERN con Predicciones mediante Machine Learning

Este proyecto es una aplicación full-stack tipo dashboard financiero, desarrollada con el stack MERN, que integra predicciones basadas en machine learning para ofrecer análisis financieros orientados a datos.
El frontend está construido con Vite como entorno de desarrollo para obtener compilaciones rápidas y recarga en caliente. Redux Toolkit se utiliza para la gestión del estado global, Material UI (MUI) proporciona una librería de componentes moderna y responsiva, y Recharts se encarga de la visualización de datos financieros mediante gráficos interactivos.
El backend está desarrollado con Node.js y Express.js, exponiendo APIs REST para la gestión de datos y la lógica del negocio. MongoDB se utiliza como base de datos para el almacenamiento persistente de información financiera y resultados predictivos. La aplicación sigue una arquitectura modular y escalable, pensada para escenarios reales de producción.

Stack Tecnológico

Frontend: Vite, React, Redux Toolkit, Material UI, Recharts
Backend: Node.js, Express.js
Base de datos: MongoDB
Machine Learning: Modelos predictivos para análisis financiero
